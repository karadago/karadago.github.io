[TBEC-2018 JOINT GAP CALCULATOR]

CODE REQUIREMENT:

According to the Turkish Building Earthquake Code (TBEC-2018), joint gaps are required (or advised) for adjacent buildings, structures with distinct height or foundation level differences, buildings with plan lengths exceeding 40m, or those featuring severe plan irregularities such as re-entrant corners (e.g., L-, T-, or U-shaped layouts).
MINIMUM REQUIRED JOINT GAP: 60 mm
